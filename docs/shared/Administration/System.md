This page displays technical information on the BCMS system.

# Control Panel tab

**Site in maintenance**: this is toggled on when admins are running maintenance onto the system. It activates a thin line with “Site in maintenance” written on for any logged in user. 

**Reset terms and conditions request**: if Enable Terms & Conditions is turned on, pressing the reset button will prompt the system to display the popup again to the same user after their first login. This is useful when several users need to all accept and might share a profile.

**Enable Terms&Conditions**: if turned on, it displays a T&C popup after their first login for all users.

**Locale Timezone**: displays date and time according to the time zone - to be set always for the same timezone used by the site. 

**Software infos **and **Changelog**: these are never used.

# Other tabs

The other tabs display the following information:

- **Server tab**: server information
- **Core tab**: software and installed version
- **Database tab**: space occupied by each database table
- **License tab**: license information
