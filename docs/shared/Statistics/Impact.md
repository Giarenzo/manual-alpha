This page provides detailed statistical insights and visualizations for impact-related events. It allows you to gain statistical insights thanks to the log of all events recorded manually in the impact form of the event management menu. User may then make data-informed safety decisions related to bird strikes.

# Functions

- In the top left corner you will see a download icon. This allows you to extract an xlsx file containing all data related to the chosen are and timeframe.
- In the top right corner, a pop-up menu will appear once you click on the calendar icon. This allows you to select a timeframe for which you want to see data. All displayed entries will refer to that timeframe.
- You will be able to select how you view the collected data according to 4 criteria: time, species, flight phase, aircraft parts.
- Caution: the quality and reliability of these data depends solely on the manual input of bird strikes. The system does not populate this database automatically.
