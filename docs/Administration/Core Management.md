This page displays all the active services running in the BCMS system. It allows users to see at a glance what is currently active or not and for what purpose, and to activate or stop a service, should that be necessary.

## The Top Bar

- In the top-right corner, the two-arrow icon resets table filters to their default value, whereas the grid icon allows you to select the columns you would like to be displayed in the list.
- Once the list is displayed, you can use the first row of the table to order, search and filter results according to the available options.

## The User List

The list can provide the following pieces of information with respect to the services, these can be displayed according to what is checked in the grid-icon pop-up menu (see above).

- **ID**: the unique identification number of the system
- **Name: **the systemâ€™s name
- **Description**: a free-text description of the service 
- **No. of panels**: this is a legacy option, it serves no practical purpose.
- **State**: the status (e.g. running, not installed) of each service 

At the end of each row, two buttons allow users to Activate or Stop each service.
