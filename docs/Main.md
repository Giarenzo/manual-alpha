- This is the homepage. It display links to all the pages in the system and it replicates the menu you would see when clicking on the burger icon. This page allows you to navigate to all the pages of the system that you have access to.

    - Go to the level 1 page and find the link from there
    - Use the side menu you can access via the burger icon (top-left)
