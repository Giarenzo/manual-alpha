# Inspection forms

In this section, you will find three different tabs (accessible by submenus) allowing you to log inspection forms. The Inspection Forms facilitate detailed documentation of inspections related to bird strikes and other relevant activities. These forms are essential for maintaining accurate records and ensuring thorough inspections.

There are three tabs:

- Site inspection
- Bird Observation Survey
- Mortality search

## Site inspection

The site inspection tab allows a user to log a physical visit to a sector via a standard form. If a physical visit to a sector is requested, this should be logged here.

- The orange plus icon in the top left-corner opens an input form. The form allows for manual input of all relevant data. Once saved, data from each saved form will appear as a single entry in the log table.
- In the top-right corner, the two-arrow icon resets table filters to their default value, whereas the grid icon allows you to select the columns you would like to be displayed in the log.
- Once the log is displayed, you can use the first row of the table to order, search and filter results according to the available options

# Form options

The different forms offer similar options, among them are the following.

## Fields for Start and End Time, Supervisor Details, Weather Conditions

**Inspection Timing and Details**: The form includes fields for recording the start and end times of inspections, supervisor details, and weather conditions, ensuring comprehensive documentation.

## Cause of Inspection

**Inspection Reason**: Users can specify the cause of the inspection, such as routine checks, incident response, or follow-up on a previous issue.

## Agricultural Activity and Bird Deterrence Measures

**Contextual Information**: The form allows for the documentation of any agricultural activities in the vicinity and the bird deterrence measures in place during the inspection.

## Add Photos and Inspection Outcomes

**Visual and Outcome Documentation**: Users can attach photos taken during the inspection and record the outcomes, including any actions taken or recommendations made.

## Inspections Description

The Inspections section focuses on the comprehensive documentation of various inspection activities. It ensures that all relevant information is accurately recorded, facilitating effective management and analysis of inspection data.
